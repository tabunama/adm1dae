adm1dae
=======

Python package for ADM1 DAE process modelling with BSM2 algebraic pH and H2 solvers.

Install from PyPI:

.. code-block:: bash

   pip install adm1dae

Current published artifact:

- Windows x64
- CPython 3.12 wheel
