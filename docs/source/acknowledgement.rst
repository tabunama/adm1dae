Acknowledgement
===============

The ``adm1dae`` package was prepared by `Dr. Taher Abunama <mailto:tabunama@gmail.com>`_, based on the original ADM1 / BSM2 implementation with algebraic pH and H2 solvers developed in the academic wastewater modelling community.

The packaged workflow follows the Matlab/Simulink ADM1 implementation notes and the later aqueous-phase chemistry / ion-speciation framework integrated with ADM1 in the BSM2 context.

Acknowledgement is due to the upstream academic contributors, including Christian Rosen, Darko Vrecko, Ulf Jeppsson, Xavier Flores-Alsina, and Krist V. Gernaey.
