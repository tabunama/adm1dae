adm1dae – Third-Party Notices / Attribution

This distribution contains compiled extensions derived from ADM1 / BSM2 wastewater process model components and associated algebraic solver components.

Included compiled components

- adm1_DAE2_bsm2 : ADM1 DAE reactor kernel
- pHsolv_bsm2 : pH algebraic solver
- Sh2solv_adm1 : dissolved hydrogen algebraic solver
